# Use venv
`python -m venv myenv`  
`.\myenv\Scripts\activate`  

# Install dependencies
`pip install websockets flask`  
`pip install flask`  
`pip install websockets`  

# Run the server
`python flask_server.py`  
`python websocket_server.py`  

or use bash scripts
### Windows  
`run_server_windows.bat`  
### Linux
`run_server_linux.sh`
